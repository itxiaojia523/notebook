    # JS简介

    web标准，XHTML和CSS通常占据核心地位。
    由W3C批准并与标准相兼容的Web浏览器支持的第三方技术成为DOM。可以利用DOM去改善文档的可交互性

    # 什么是DOM

    文档对象模型，一套对文档的内容进行抽象和概念化的方法

DOM是一种API 应用编程接口。 API是一组已经得到各方共同认可的基本约定。
现实中的API的例子：莫尔斯码，国际时区，化学元素周期表

现代浏览器至少实现了W3C相关标准中的95%的规范。
意味着JS代码几乎不需要再考虑它们将运行在何种浏览器环境下了

    # DHTML

    dynamic HTML
1. 刘勇HTML把网页标记为各种元素
2. 利用CSS设计元素排版和样式
3. 利用JS操控和改变有关样式

    # 对象分类

    user-defined object,自定义
native object 内建在JS中的对象，如Array，Math，Date等
host object 宿主对象：浏览器提供的对象

宿主对象指Window, 对应着浏览器窗口本身
这个对象的属性和方法通常被称为BOM 
window.open window.blur 往上冲浪时各种弹窗
