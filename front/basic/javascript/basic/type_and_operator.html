    数据类型

    # 数据类型

    值类型(基本类型)：字符串（String）、数字(Number)、布尔(Boolean)、空（Null）、未定义（Undefined）、Symbol。

    引用数据类型（对象类型）：对象(Object)、数组(Array)、函数(Function)，还有两个特殊的对象：正则（RegExp）和日期（Date）。

    # 基本类型

    1.String
2.Number
3.Boolean
4.Null 空，不存在
5.Undefined 未定义，存在这个变量但是没值
6.Symbol 忘记了

    # 类型转化

    	转为字符串：''
	转为数值：parseInt();parseFloat();- * /
	转为布尔值：Boolean()

    var year = prompt("请输入您的出生年份:") 
    //prompt获得的是String，想当做Number用来计算，要将String转换为Number
    var age = 2021 - year; //减法隐式转化
    alert("您今年"+age+"岁") //字符串拼接  String+其他，其他会被隐式转换成String

    var num1 = prompt("请输入第一个值：")
    var num2 = prompt("请输入第二个值：")
    var result = parseFloat(num1) + parseFloat(num2) //prompt的返回值类型是字符串
    alert("计算的结果是"+ result)

    var name = prompt("请输入姓名：")
    var age = prompt("请输入年龄：")
    alert("名字:"+ name+"\n年龄："+age) //反斜杠n要放在引号中，实现换行

    # 基本包装类型：String Number Boolean

    对象才有属性和方法

    	var str = 'andy'
	console.log(str.length) //4
    //对象才有属性和方法,为什么str会有length属性呢
    //隐藏机制
    相当于	var temp = new String('andy') //包装
	str = temp //赋值
	temp = null //销毁临时变量

    # 字符串操作方法（重点）

    注意：字符串所有的方法，都不会修改字符串本身，
操作返回一个新的字符串（开辟一个新的内存空间，存放字符串）

        // 拼接字符串，+号更常用
	concat(str1,str2)
    // 截取字符串
	substr(start,length)//从start开始，截取length个字符 重点！
	slice(start,end) // end取不到
	substring(start,end) //end取不到 基本同slice,但不接受负值
	// 替换字符
	replace('被替换','替换') //只替换第一个
    // 把字符串转为字符串数组；
	split('分隔符') 
    ex.
	var str3 = 'red&pink&blue'
    str3.split('&') //["red", "pink", "blue"]
    //join（）把数组转为字符串

	toUpperCase()//转为大写
	toLowerCase()//转为小写

	indexOf('要找的字符',[起始位置])// 根据字符返回位置; []表可选，没有则返回-1
	charAt(index) //根据位置返回字符
	charCodeAt(index) //根据位置返回字符的ASCII码
	str[index] //IE8+

    # 运算符

    # 算术运算符

    	%：取余 主要用来判断一个数是否能被整除 余数为0
    ++:
	前置递增：先+1，再返回值
	后置递增：先返回原值，再+1;即 a =2; a++ =1;（用更多）

    # 比较运算符

    	console.log(18 == '18') true 会转型，转型后相等也相等
	console.log(18 === '18') false 表全等 一般用这个

    # 逻辑运算符

    	&& || !
	*（重要）短路运算（逻辑中断）：当有多个表达式（值）时，左边的表达式可以确定结果时，就不再继续运算
	1.逻辑与 语法：表达式1 && 表达式2
	如果表达式1的值为true，则返回表达式2
	如果表达式1的值为false,即可确定结果，则返回表达式1
