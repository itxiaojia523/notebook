    # 标签属性

    <marquee loop="1" bgcolor="red">
	hello world
	<input type="password" disabled>
</marquee>

以上效果是 只循环一次；背景颜色为红色

释意
1. marquee这个标签 有loop这属性
2. = 是赋值符号 不是相等符号
属性名 = 属性值： 把值赋值给属性 而不是相等
3. 为什么这里能写loop 因为marquee这个标签，是内置标签（已经定义好了的标签，不是用户我自定义的标签），这个内置标签里已经定义好了很多属性，所以属性名等不能随便写
怎么看能写什么，百度它的官网或者直接百度

暂时理解到这里

3. "" 表示字符串类型
4. "1" 和 1 的区别在于 前者是字符串类型 后者是 数字类型
5. loop = "1" 的意思是 把"1"(字符串的1) 赋值给 loop 
6. 为什么不是 loop = 1 是因为 loop这个属性 需要 字符串类型 而不是数字类型

关键词： 属性 赋值 类型 值

注意：
不同标签有不同属性
也有一些属性时通用的 之后总结，用多了就知道了

		• Web Storage API：sessionStorage(保存在session中，浏览器关闭，数据消失)、localStorage(保存在本地
		//sessionStorage和localStorage方法一致

		localStorage.setItem(key,value);

		localStorage.getItem(key);

		• Web Workers API：Web Workers可以让Web应用程序具备后台处理能力，对多线程的支持性非常好。
		但是在Web Workers中执行的脚本不能访问该页面的window对象，
		也就是Web Workers不能直接访问Web页面和DOM API。
		虽然Web Workers不会导致浏览器UI停止响应，但是仍然会消耗CPU周期，导致系统反应速度变慢。
		新开子线程，实际还是单线程
			用于大数据处理 或 耗费时间较长的操作

	2.返回到之前的页面，页面的状态和数据会丢失
方案：表单只负责采集，ajax负责提交数据

    function addEventListener(element,eventName,fn){
      if (element.addEventListener){
        element.addEventListener(eventName,fn)
      }else if(element.attachEvent){
        element.attachEvent('on'+eventName,fn)
      }else{
        element['on'+eventName] = fn

缓动动画
	最常见：让速度慢慢停下来
1.盒子每次移动距离慢慢变小
2.算法：（目标值-现位置）/10 为步长
3.停止条件：目标位置
注意：步长取整
var step;
    // 简单动画函数封装
    function move(obj, target, callback) {
      //通过属性赋值的方法，实现给不同元素指定不同的定时器，优化var开辟新空间
      clearInterval(obj.timer) //先清除定时器，避免重复叠加
       obj.timer = setInterval(function (){
        if (obj.offsetLeft >= target) {
          clearInterval(obj.timer)
          callback();
        }
        //步长值,涉及到除法，可能除不断，到不了目标位置比如500，实际494.6，将step取整
        // 正值往上取整，倒退应该往下取整 Math.floor
         step = (target - obj.offsetLeft / 10);
        step = step > 0? Math.ceil(step) : Math.floor(step)
        obj.style.left = obj.offsetLeft + step + 'px'
      },15)
    }
