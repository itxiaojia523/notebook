    # 使用 Markdown 记笔记

    https://zhuanlan.zhihu.com/p/496076040

    # 工具

    # Visual Studio Code

    默认支持 Markdown 语言，只需要打开一个 .md 后缀的 Markdown 文件，再按下快捷键 Ctrl + K + V 就可以显示。并且，通过一些其他插件，如 Markdown Preview Enhanced 和 Markdown All in One，就可以有更好的 Markdown 编辑体验。如果你需要「所见即所得」功能，也可以试试 Milkdown 插件。

    # Typora

    Typora 也是一款很受欢迎的 Markdown 编辑器。有一个巨大的缺点，就是它并不是免费的。
